# Visual Sorter

An interactive sorting algorithm visualizer built with **React + Vite**.

## Algorithms
Bubble · Selection · Insertion · Merge · Quick · Counting Sort

## Features
- Real-time bar chart animation
- Step-by-step mode
- Code in C, C++, Java, Python
- Time & space complexity details
- Light / Dark theme
- Algorithm performance comparison

## Run Locally
\`\`\`bash
npm install
npm run dev
\`\`\`
