#  Visual Sorter – Interactive Sorting Visualizer

An interactive **sorting algorithm visualizer** built using **React + Vite**, designed to help users understand how different sorting algorithms work through real-time animations and comparisons.

---

## 🌐 Live Demo

👉 https://tamohar20.github.io/visual-sorter-react/

---

## 🧠 Algorithms Implemented

* 🔵 Bubble Sort
* 🟢 Selection Sort
* 🟡 Insertion Sort
* 🔴 Merge Sort
* 🟣 Quick Sort
* 🟠 Counting Sort

---

## ✨ Features

* 📊 Real-time bar chart visualization
* ⏯️ Step-by-step execution mode
* 💻 Code implementation in multiple languages:

  * C
  * C++
  * Java
  * Python
* ⏱️ Time & Space Complexity display
* 🌗 Light / Dark theme toggle
* ⚖️ Algorithm performance comparison

---

## 🛠️ Tech Stack

* ⚛️ React (Frontend UI)

* 🎨 CSS (Styling & animations)

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Tamohar20/visual-sorter-react.git
cd visual-sorter-react
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run locally

```bash
npm run dev
```

---

## 📦 Build for Production

```bash
npm run build
```

---

## 🌍 Deployment

This project is configured for GitHub Pages deployment using Vite’s base path:

```js
base: "/visual-sorter-react/"
```

(as seen in vite.config.js )
